#### PHP Array 简介
array 函数允许您对数组进行操作。


PHP 支持单维和多维的数组。同时提供了用数据库查询结果来构造数组的函数。








#### 安装
array 函数是 PHP 核心的组成部分。无需安装即可使用这些函数。
