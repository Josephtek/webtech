#### jQuery 图标

 在 jQuery Mobile 中，如需为按钮添加图标，请使用 data-icon 属性：

 
```
  <a href="#anylink" data-role="button" 

#### data-icon="refresh"

>Refresh Page</a>

 


```
 

#### 提示：

在 <button> 或 <input> 元素中，您也可以使用 data-icon 属性。
