 CSS 尺寸 (Dimension)  
CSS 尺寸 (Dimension) 属性允许你控制元素的高度和宽度。同样，它允许你增加行间距。

#### 所有CSS 尺寸 (Dimension)属性

 

|属性|描述|
|:--|:--|
|height|设置元素的高度。|
|line-height|设置行高。|
|max-height|设置元素的最大高度。|
|max-width|设置元素的最大宽度。|
|min-height|设置元素的最小高度。|
|min-width|设置元素的最小宽度。|
|width|设置元素的宽度。|
