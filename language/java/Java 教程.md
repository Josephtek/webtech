#### 我的第一个JAVA程序

  以下我们通过一个简单的实例来展示Java编程，本实例输出"Hello World"，这也是所有语言入门的第一个实例程序：

 
```

public class MyFirstJavaProgram {

    public static void main(String []args) {
       System.out.println("Hello World");
    }
} 

```
